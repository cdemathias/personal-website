# charlesmathias.com

Personal website for Charles Mathias — a single-page, non-scrollable static site.

## Stack

- Plain HTML & CSS, no frameworks or dependencies
- Google Fonts: Cormorant Garamond + EB Garamond

## Local preview

Open `index.html` directly in a browser, or run a local server:

```bash
npx serve .
```

## Deployment

Deployable to any static host. For GitHub Pages, push to the `main` branch and enable Pages in the repository settings (set source to `/` root).
